Background Generator

This background generator is built using HTML, CSS and JavaScript. First, I created the basic layout of the app with HTML. Then I added 2 color inputs, followed by styling the page using CSS. Next, I started creating my JavaScript file. Using DOM Manipluation, I added the document.querySelectors to control the color inputs, added EventListeners, then created the function to change the gradient when each input is selected. Lastly, I added css.textContent to display the gradient colors to easily be able to insert the gradient colors into your project.
